# Bookstore CRUD Web Application

A **full-stack MERN (MongoDB, Express, React, Node.js) application** for managing books. Users can **create, read, update, and delete (CRUD)** books with details like name, author, price, and publish date.

---

## Features

- Add a new book with:
  - Book Name
  - Book Title
  - Book Author
  - Selling Price
  - Publish Date
- Edit existing book details
- Delete books
- View all books in a structured table
- Interactive frontend with React
- Backend API built with Node.js and Express
- Data stored in MongoDB
- Axios used for frontend-backend communication

---

## Technologies Used

- **Frontend:** React.js, Axios, HTML, CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Version Control:** Git & GitHub
